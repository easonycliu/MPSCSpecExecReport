# Compare SATA and nvme
## Settings
* Memory limit: 32G
* Osprey param: window size 8192, batch size 32768
* mem limit low = 24G, mem limit high = 28G, mem limit max = 32G
* Vector size: 512
* Thread num: 1
* SATA: 500 MB/s
* nvme: 2 GB/s

| mvmul     |  169.6s     |  89.6s      |  161.4s   |  161.2s   |
